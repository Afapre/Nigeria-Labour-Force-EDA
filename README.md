# Nigeria Labour Force EDA Dashboard

## Project Overview
This project presents an exploratory data analysis (EDA) of the Nigeria Labour Force Survey dataset. The goal is to uncover key trends in employment, unemployment, and workforce composition to support data-driven decision-making for policymakers and business leaders.

---

## Objectives
- Analyze employment and unemployment patterns
- Identify demographic and regional disparities
- Explore relationships between education and employment
- Provide actionable insights based on data
